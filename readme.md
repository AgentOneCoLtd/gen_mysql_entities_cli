# gen_mysql_entities_cli

generate most used mysql entity functions in our codebase

## install

```bash
$ npm install -g @ag1/gen_mysql_entities_cli
```

## usage

```bash
$ gen-mysql-entities -o <outputDir> -h <host> -p <port> -u <user> -x <password> -d <database>
```
